## Offline Reinformcement Learning with Reservoir Computing 
This repo contains implimnetations of reservoir computing for offline reinforcement learning for continuous control task

## Installation
Requires python >= 3.10
```bash
pip install -r requirements.txt

git clone https://github.com/stefanonardo/pytorch-esn.git
cd pytorch-esn
pip install -e ./

```

Dataset links for d4rl: https://github.com/Farama-Foundation/d4rl/wiki/Tasks
Environment info for mujoco: https://gymnasium.farama.org/environments/mujoco/
Environment info for robotics: https://robotics.farama.org/
